📌 Project Overview

This project focuses on analyzing Ecommerce sales data using MySQL.
The goal is to extract meaningful business insights such as revenue trends, top customers, best-selling products, and customer purchase behavior using SQL queries.

This project simulates a real-world Ecommerce database with structured tables and performs analytical queries that are commonly used by Data Analysts.

🗂️ Database Schema

The database consists of three main tables:

1️⃣ Customers Table
Column Description
customer_id Unique ID for each customer
customer_name Name of the customer
city Customer location

2️⃣ Products Table
Column Description
product_id Unique product ID
product_name Name of the product
category Product category

3️⃣ Orders Table
Column Description
order_id Unique order ID
customer_id Foreign key from customers
product_id Foreign key from products
order_date Date of purchase
quantity Number of units purchased
price Price per unit

🔗 Entity Relationship

One customer → Many orders

One product → Many orders

Orders table acts as a fact table for analysis


📊 Business Problems Solved

The following analytical questions were answered using SQL:

✔ Total Revenue

Calculated total sales revenue using:

SUM(quantity \* price)

✔ Monthly Sales Trend

Identified revenue generated per month to analyze sales growth.

Business Use:
Helps in understanding seasonal trends and forecasting.

✔ Top 5 Customers by Revenue

Ranked customers based on total spending.

Business Use:
Useful for loyalty programs and targeted marketing.

✔ Top Selling Products

Identified products with highest quantity sold.

Business Use:
Helps in inventory planning and demand forecasting.

✔ Category-wise Revenue

Calculated revenue generated by each product category.

Business Use:
Helps management focus on high-performing categories.

✔ Repeat vs New Customers

Classified customers based on number of orders.

Business Use:
Helps in measuring customer retention.

🧠 Key Insights

Electronics category generated the highest revenue

A small number of customers contributed to a large portion of sales

Repeat customers placed multiple orders, indicating retention

Monthly revenue shows a gradual increasing trend
